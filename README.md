# FPGA-Based Digital Control System
## Overview
A digital logic controller designed in Verilog HDL for sensor-actuator management. Implemented using Xilinx Vivado.

## Key Features
* **FSM Implementation:** 4-state Mealy Machine for mode control.
* **Testbench Verified:** 100% functional coverage in ISim.
* **Modular Design:** Parameterized Verilog modules for reusability.
